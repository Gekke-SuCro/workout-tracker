# Auth Validation Rules

## Login

- **Username & Password**:
    - required

## Registration

- **First Name & Last name**:
    - required
    - 2-50 chars
    - only alphabets
    - no whitespaces


- **Username**:
    - required
    - unique
    - 3-30 chars 
    - first char must be alphabetic
    - no whitespaces


- **Password**:
    - required
    - 8-128 chars


- **Confirm Password**: Must match Password.
